# my_chat_app

#### 介绍
一个基于django、nodejs、vue的websocket实时点对点通讯项目，关键词 QQ、微信、实时聊天、django、vue、nodejs、websocket。

#### 软件架构
本项目完成的功能：注册登录、搜索添加好友、好友之间点对点实时通讯、信息查看、历史消息记录。
项目http服务端又django提供
项目websocket服务端由nodejs提供
项目前端部分由vue提供
项目主要数据库MySQL
